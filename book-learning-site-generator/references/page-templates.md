# Page Templates

## Homepage

Required sections:

1. Hero
   - book title
   - author
   - one-sentence learning promise
   - start/continue buttons
2. Book facts
   - language
   - chapter count
   - type
   - recommended learner
3. AI recommended learning path
4. Full book map
   - parts
   - chapters
   - sections overview
5. Precision reading guide
   - intensive/skim chapters
6. Quick entries
   - formulas
   - glossary
   - learning paths
   - modern extensions if enabled
7. Progress/memory panel if enabled

## Chapter Page

Required order:

1. Chapter header
   - original number/title
   - one-sentence summary
   - original page/section location
2. Chapter orientation
   - core problem
   - learning goals
   - prerequisites
   - position in the whole book
3. Original-book alignment
   - original section list
   - mapped concepts
   - reading recommendation
   - intensive/skim notes
4. Core concepts
   - location
   - problem
   - intuition
   - formal expression
   - relation
5. Formulas or key definitions
6. Visual/interactive module if useful
7. Examples, analogies, or cases
8. Common misconceptions
9. Knowledge connections
10. Key Takeaways
11. Reading checklist
12. Modern extensions, clearly labeled if enabled
13. Feynman restatement / chapter memory
14. Previous/next chapter navigation

Chapter memory belongs near the end, after the user has enough context to summarize.

## Formula Reference Page

Each card:

- formula name
- LaTeX
- chapter/section
- symbol table
- intuition
- use case
- common mistake
- related concepts
- source label

Filters:

- chapter
- category
- source
- difficulty

## Glossary Page

Each item:

- Chinese term
- English term
- chapter/section
- short definition
- original-book context
- related terms
- high-frequency flag

## Learning Paths Page

Include paths based on book diagnosis:

- quick scan
- careful reading
- interview/exam review
- practical application
- beginner path
- research path

Each path:

- audience
- chapter order
- skip advice
- estimated time
- stage goals
- completion checklist

## Modern Extensions Page

Only include if enabled. Clearly state:

- not original book content
- why it is relevant
- which original chapters it extends
- what to read after the original chapter

Never insert modern material into original chapter summaries as if it were in the book.
